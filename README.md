## Installation

To get started, follow these steps:

1. Clone the repository:
   ```bash
     git clone https://github.com/4ssh1/TODO_LIST.git
   ```
   
2. Navigate to the project directory:
   ```bash
     cd TODO_LIST
   ```

4. Install dependencies:
   ```bash
     npm install
   ```

6. Start the server:
   ```bash
     npm run dev
   ```
